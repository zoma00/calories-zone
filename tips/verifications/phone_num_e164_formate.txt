Phone Number in E.164 Format

Short answer
+ followed by country code and full phone number, with no spaces, no dashes, and no leading zero.

Basic rule
- Start with +
- Add the country code
- Add the national phone number
- Remove spaces, brackets, dashes, and the leading local 0 if your local format uses one

Format example
+201234567890

Common examples
- Egypt: 01234567890 becomes +201234567890
- Saudi Arabia: 0501234567 becomes +966501234567
- United States: (415) 555-2671 becomes +14155552671
- United Kingdom: 07123 456789 becomes +447123456789

What not to submit
- 01234567890
- 00201234567890
- +20 123 456 7890
- +20-123-456-7890

For Google forms or Play Console
Use the full international number in one line, for example:
+201234567890
